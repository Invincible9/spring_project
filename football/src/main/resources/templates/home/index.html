How to use it:

1.Firstly, you need to create nationality, for example - England, Italy, Spain, Germany, France.. etc

2.Next step is to create division for example - Premiership, Seria A... etc

3.After creation of the nationality and division, now you must create teams, for example Liverpool, Parma, Barcelona, Bayern.. etc

4.Next you must create seasons - for example 2020-2021, 2019-2020 or maybe season 2020-2021 etc..

5.Next you must create rounds, for example - round 01, round 02 or maybe just 01, 02, 03 etc...

6.Final step is to create game - Arsenal - Man Utd for example or something like that.
